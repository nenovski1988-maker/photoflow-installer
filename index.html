PhotoFlow

Инсталация с автоматични ъпдейти (App Installer).
На Windows 10 е нужно еднократно да се инсталира сертификатът, иначе ще видиш 0x800B010A.

1) Сертификат (само първия път)
Свали сертификат (.cer)
2) Инсталация / Update
Инсталирай / Обнови PhotoFlow
Ако бутонът “Install” е сив:
  • Инсталирай сертификата: double click на .cer → Install Certificate → (най-сигурно) Local Machine → Trusted People.
  • Провери, че няма стара инсталация с друг Publisher (деинсталирай PhotoFlow ако има).

Директен файл (ако някой браузър блокира протокола):
PhotoFlow.Package.appinstaller

Лиценз файл: %USERPROFILE%\Documents\PhotoFlow\Licenses\photoflow.license.json
